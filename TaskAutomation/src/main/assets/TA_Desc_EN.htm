·           Always hold Wake Lock

·           Setting individually

-/mnt/sdcard/TaskAutomation/TaskAutomation_log_YYYY-MM-DD_HH-MM-SS-TTT.       txt(Archive log file)

b.    Debug message level

Set the debugging information. Please set to "0" usually. If you are debugging a BeanShell script, method result of TaCmd is recorded If you specify "2".

-0 No debug message

-1 Debug message

-2 Detail debug message1

-3 Detail debug message2

c.  Max log file count

    Specifies the maximum number of log files. The default value is 10.

d.  Log file directory
